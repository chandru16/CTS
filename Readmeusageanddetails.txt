/*Use Of README*/

* A Readme file is a file that describes the purpose of the repository and gives hints on what the code does,
  how to compile/use it, etc.

* The .md extension stands for Markdown, which is a type of file that is both readable in plain text, 
  but can also be easily converted to HTML to display special elements

* We can generate links to other pages and also other README Files.
